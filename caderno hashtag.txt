Emmet: Wrap with abreviation (Encapsular com uma tag)

! - Cria automaticamente o começo do html

TAG DE TEXTO: 
H - Tag de título
p - Tag de paragrafo
span - Tag de paragrafo que não quebra linha 
b - tag de negrito
strong - tag de negrito
dl/dt/dl - tag para criar titulos dentro do paragrafo
em - tag de itálico
code - tag para formatação de código
blockquote - comentarios no meio 

OUTRAS TAG
hr - coloca uma linha horizontal na página
br - quebra a linha
<p id="" - Tag que define um id para o paragrafo ou titulo
figure - seção para imagem
img - adiciona imagem
a - adiciona hiperlink
ol / ul - Lista ordenada - lista não ordenada
li - linha de lista
